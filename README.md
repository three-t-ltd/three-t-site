# three.T Website — Kinetic Edition (v2)

ビルド不要の静的ホームページ。v1の構成・コピーを引き継ぎつつ、より動的な演出に全面リニューアルした版です。

## v1からの主な変更点（動的演出）

- プリローダー（カウントアップ + ロゴ + スライドアウト）
- カスタムカーソル（リング + ドット、ホバーで拡大）※PCのみ
- ヒーロー背景：Canvasパーティクルネットワーク（前方へ流れる粒子 + マウス視差）
- ヒーロー見出し：1文字ずつのスプリットテキスト出現 + 「進む」のグリッチ／スクランブル演出
- 画面上部にスクロール進捗バー
- セクション間のキネティック・マーキー（2本、逆方向）
- ストーリーセクション：PCはスクロール連動の横スクロールパネル、モバイルは縦タイムライン
- カード全般：3Dチルト + カーソル追従グロー
- ボタン：マグネティック（カーソル吸着）+ クリップパスホバー
- 実績数値カウントアップ、プロセスのライン描画、フッターの巨大アウトラインロゴ
- `prefers-reduced-motion` 完全対応（全演出を無効化）

## プレビュー

```powershell
node preview-server.mjs
```

ブラウザで `http://127.0.0.1:4173` を開きます。

## 公開前の差し替え

1. `assets/portrait.jpg` を高解像度の写真に差し替える（現在はnoteのプロフィール画像330pxを仮使用。同じ写真の高解像度版を上書き保存すればOK）
2. 会社概要、プライバシーポリシーのリンク先を設定する
3. フォームの `data-endpoint=""` に送信先APIのURLを設定する（GitHub Pagesの場合はFormspree等）
4. 構造化データへ公式URL、所在地、連絡先を追加する
5. OGP画像とfaviconを追加する

フォームは入力検証と送信処理を実装済み。送信先API未設定の場合は、入力確認後に設定が必要であることを画面へ表示します。
